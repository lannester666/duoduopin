1. 登录

登录使用的是仿JWT方式。在成功登录后会返回 ididid 和 tokentokentoken 。在之后进行需要登录才能使用的功能时，要将HTTP请求的 authorizationauthorizationauthorization 头内容设置为 {id}_{token}\{id\}\_\{token\}{id}_{token} 的格式。不确定以后会不会更改，先凑合着用。

1.1 注册

content−type:multipart/form−datacontent-type: multipart/form-datacontent−type:multipart/form−data
参数：
usernameusernameusername : 用户名
nicknamenicknamenickname : 昵称
passwordpasswordpassword : 密码

PUTPUTPUT 方式发送请求至

http://123.57.12.189:8080/User/register

1.2 登录

content−type:multipart/form−datacontent-type: multipart/form-datacontent−type:multipart/form−data
参数：
usernameusernameusername : 用户名
passwordpasswordpassword : 密码

POSTPOSTPOST 方式发送请求至

http://123.57.12.189:8080/User/login

返回结果：
ididid : 用户 ididid
tokentokentoken : 令牌，有效期为 727272 小时

1.3 登出

authorizationauthorizationauthorization : [用户 ididid] + "_" + tokentokentoken

DELETEDELETEDELETE 方式发送请求至

http://123.57.12.189:8080/User/logout

登出后令牌有效期清零，无法使用。

2. 拼单

以后可能会添加分页。按序排列在分页之后实现。

2.1 添加拼单

content−type:multipart/form−datacontent-type: multipart/form-datacontent−type:multipart/form−data
authorizationauthorizationauthorization : [用户 ididid] + "_" + tokentokentoken
参数：
typetypetype : [MEALMEALMEAL, CARCARCAR, HOUSEHOUSEHOUSE, OTHEROTHEROTHER]，分别对应拼餐、拼车、拼房、其他
descriptiondescriptiondescription : 描述
addressaddressaddress : 详细地址
timetimetime : 时间，以 yyyy−MM−ddhh:mm:ssyyyy-MM-dd hh:mm:ssyyyy−MM−ddhh:mm:ss 格式发送（秒数可以统一设置为 000000）
curPeoplecurPeoplecurPeople : 当前人数（默认为 111）
maxPeoplemaxPeoplemaxPeople : 最大人数
pricepriceprice : 预计价格
longitudelongitudelongitude : 经度（通过地图 APIAPIAPI 获取，尽量精确）
latitudelatitudelatitude : 纬度

PUTPUTPUT 方式发送至

http://123.57.12.189:8080/ShareBill/add

2.2 查找拼单

2.2.1 按拼单id查找

POSTPOSTPOST 方式发送至

http://123.57.12.189:8080/ShareBill/{id}

使用拼单 ididid 代替路径上的 {id}\{id\}{id} 即可

2.2.2 按用户id查找

POSTPOSTPOST 方式发送至

http://123.57.12.189:8080/ShareBill/user/{id}

使用用户 ididid 代替路径上的 {id}\{id\}{id} 即可

2.2.3 按其他信息查找

content−type:multipart/form−datacontent-type: multipart/form-datacontent−type:multipart/form−data
参数：
typetypetype : 类型，同上
descriptiondescriptiondescription : 关键词，多个关键词之间使用空格隔开
startTimestartTimestartTime : 开始时间，格式同上

POSTPOSTPOST 方式发送至

http://123.57.12.189:8080/ShareBill/info

DELETEDELETEDELETE 方式发送至
